# Portafolio Odoo & Integraciones

Estructura:
- index.html: Página principal
- assets/css/style.css: Estilos base
- assets/js/main.js: Lógica (carga dinámica, filtros, tema)
- data/*.json: Fuentes de datos (proyectos, integraciones)

Deploy rápido (ej. Netlify):
1. Subir repositorio.
2. Configurar build (estático).
3. Activar formularios (Netlify Forms).

Personalización:
- Agregar más proyectos en data/projects.json
- Ajustar paleta en :root (style.css)
- Incluir imágenes en assets/img y referenciar dentro de tarjetas.

Mejoras futuras:
- Internacionalización
- Modo offline
- Tests de accesibilidad
